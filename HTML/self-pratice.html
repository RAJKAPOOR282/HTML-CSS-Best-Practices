Frontend Development

Frontend development is the development of the grafical user interface of a website,through the use of HTML,CSS and Javascript,so that users can view and interact with that website

  • HTML: The Hyper markup language or HTML is the standard markup language for document designied to be display in a web browser.

  • CSS: Cascading Style Sheet(CSS) is a style sheet language used for describing the presentation of a document written in a language such as HTML or XML.
  1. Pure CSS
  2. Bootstrap(Framework)
  3. Tailwind(Framework)
  4. Sass(Preprocessor)
  • JavaScript: JavaScript often abbreviated JS, is a programing language that is one of the core tehnologies of the wide web, alongside HTML and CSS. It is use to add functionality in the website.
  1. Vanilla JS
  2. jQuery(Library)
  3. React(Framework)
  4. Angular(Framework)
  5. Vue(Framework)
  6. Typescript(Preprocessor)
